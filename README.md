# 🏫 منصة الخطة التشغيلية المدرسية

منصة إلكترونية متكاملة لإدارة ومتابعة الخطة التشغيلية للمدارس السعودية، مبنية بـ HTML + CSS + JavaScript فقط.

---

## 🚀 طريقة تشغيل المشروع

### تشغيل محلي
1. قم بتحميل أو استنساخ المشروع
2. افتح ملف `index.html` مباشرة في المتصفح (Chrome أو Edge أو Firefox)
3. لا يحتاج إلى أي تثبيت أو خادم

```
افتح: index.html
```

### تشغيل عبر VS Code (اختياري)
1. ثبّت امتداد **Live Server**
2. انقر بزر الفأرة الأيمن على `index.html`
3. اختر "Open with Live Server"

---

## 📤 رفع المشروع على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub
2. ارفع جميع الملفات:
   ```bash
   git init
   git add .
   git commit -m "initial commit"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO.git
   git push -u origin main
   ```
3. في إعدادات المستودع → **Pages** → اختر `main` branch → `/ (root)`
4. سيكون المشروع متاحاً على: `https://USERNAME.github.io/REPO/`

## 💾 تخزين البيانات

> ⚠️ **تنبيه مهم:** جميع البيانات محفوظة في **LocalStorage** في المتصفح فقط.
> - لا يوجد خادم أو قاعدة بيانات خارجية
> - البيانات تبقى بعد إغلاق المتصفح
> - مسح بيانات المتصفح سيحذف جميع البيانات
> - البيانات غير مشتركة بين أجهزة مختلفة

---

## 🛠️ التقنيات المستخدمة

- **HTML5** — هيكل الصفحات
- **CSS3** — التصميم مع متغيرات CSS
- **Vanilla JavaScript** — جميع الوظائف والتفاعل
- **Canvas API** — الرسوم البيانية (بدون مكتبات خارجية)
- **LocalStorage** — حفظ البيانات محلياً
